# Air Doodle — Draw in the Air! ✨

Air Doodle is a magical web application that lets you draw colorful, glowing art in the air using your hands! Built with MediaPipe Hand Landmarker, it's a touchless creative experience perfect for kids and adults alike.

🚀 **Live Demo**: [Hosted on Vercel](https://github.com/peooozz/Air-Doodle) (Check repository settings for actual URL)

## ✨ Features

- ☝️ **One-Finger Drawing**: Point your index finger to create smooth, glowing lines.
- ✌️ **Two-Finger Hover**: Use index + middle fingers to move your cursor without drawing.
- 🖐️ **Three-Finger Eraser**: Wipe away lines with three fingers up.
- 🪄 **Shape Auto-Correction**: Draw a circle, rectangle, or triangle and hold it still for half a second to snap it to perfection!
- 🤏 **Pinch-to-Move**: Grab any drawing by pinching near it and drag it anywhere on the canvas.
- 🎨 **Kid-Friendly UI**: A vibrant, professional interface with easy-to-use color buttons.
- 🌙 **Themes**: Toggle between dark and light modes for different drawing styles.
- 🧈 **Ultra-Smooth**: Built with Exponential Moving Average (EMA) stabilization and Quadratic Bezier curves.

## 🛠️ Tech Stack

- **MediaPipe Hands**: AI-powered hand tracking.
- **Vite**: Modern frontend build tool.
- **Vanilla JS**: Real-time canvas manipulation.
- **Electron**: Compatible for desktop application use.

## 🚀 Getting Started

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/peooozz/Air-Doodle.git
   cd Air-Doodle
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

### Desktop App (Electron)

To run as a desktop application:
```bash
npm run desktop
```
